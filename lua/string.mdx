# String

1. String was stored by 8 bits.

```lua
-- note; with UTF-8
print(require("string").len("我")) -- euqal to print(#"我") => 3

print(require("utf8").len("我")) -- => 1
```

2. String expression

```lua
------------------------------------
-- Singe line --
------------------------------------
s1 = 'string1'
s2 = "string2"

------------------------------------
-- Multiple line --
------------------------------------
s3 = [[
Multiple
Line
]]

-- With pairs(在第一個`[[]]`放入相同數量的`=`)
s4 = [[======[[abcd]]======]]


------------------------------------
-- 字串模板格式化
------------------------------------
require("string").format("some string")

------------------------------------
-- 字串長度
------------------------------------
print(#"abc") -- => 3

print(require("string").len("abc")) -- => 3

-- UTF-8 text
print(require("utf8").len("亞瑟")) -- => 2

------------------------------------
-- 字串連接
------------------------------------
print("亞" .. "瑟")
```
