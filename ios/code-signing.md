# Code Signing in iOS Development

## **一、iOS 上架所需的主要檔案**

1. **證書檔案**

   - **.csr** (Certificate Signing Request）

   - **.cer** (Certificate)

   - **.p12**（Personal Information Exchange）

   - **.pem**（Privacy-Enhanced Mail）

   - **.pkey**（Private Key）

2. **Provisioning Profile**

   - **.mobileprovision**

3. **應用程式檔案**

   - **.ipa**（iOS App Package）

4. **其他必要資源**

   - **App Icons 和截圖**

   - **App Store Connect 所需的元數據(metadata)**
